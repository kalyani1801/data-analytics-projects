# Customer Segmentation

## Overview

This project focuses on analyzing customer data and grouping customers based on shared characteristics and behavior patterns.

## Objectives

* Identify customer segments
* Analyze customer behavior
* Improve targeted marketing strategies
* Support customer-centric decision making

## Tools Used

* Excel
* Python
* Power BI

## Outcomes

* Categorized customers into meaningful groups
* Identified purchasing patterns and trends
* Generated insights for business growth
